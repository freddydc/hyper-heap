# Hyper Heap

This project contains the Next JS course

## Getting Started

Basically clone the repository:

```bash
git clone git@github.com:freddydc/hyper-heap.git
```

Install dependencies:

```bash
yarn
```

Start the project:

```bash
yarn dev
```

## Found a bug or improvement?
